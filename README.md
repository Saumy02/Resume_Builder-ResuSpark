## 🌟 Visual Features & Upgrades

ResuSpark has been transformed from a basic form into an advanced, interactive design dashboard.

### 🎨 1. Premium Light Theme Workspace
* **Glassmorphic Editor Header**: Translucent white panels (`rgba(255, 255, 255, 0.85)`) with saturation filters, smooth dropshadows, and glowing violet focus borders.
* **Canvas Grid Workspace**: A light-mode document layout desk (`#f1f5f9`) with soft dotted grids where the A4 sheet floats dynamically.
* **Modern Rounded Capsule Tabs**: Elegant, Linear/Vercel-inspired tab bar controls for switching between Templates, Content, and Design tabs.

### ✍️ 2. Canva-Style Inline Selection Toolbar
* **Highlight to Format**: Simply highlight/select any text directly on the A4 preview sheet.
* **Floating Context Menu**: A popup toolbar will appear right above your selected text containing:
  * **Bold** (𝐁)
  * **Italic** (*𝘐*)
  * **Underline** (U̲)
  * **Bullet List** (•=)
* **Real-time State Sync**: Direct changes are compiled and synchronized back to the sidebar input controls and saved automatically.

### 🫳 3. Drag-and-Drop Columns & Reordering
* **HTML5 Draggable Sections**: Drag sections (e.g., Skills, Experience, Projects) between the left and right columns of the sheet.
* **Flexible Reorder Control**: Use the floating arrow capsules (⏶/⏷) on any section or drag the handles to shift order vertically in real time.
* **Smart Headers**: Headings automatically morph their styles (underlined/border-left) depending on the column they are dropped into.

### 📂 4. Preloaded Profiles with Avatar Headshots
* **Instant Templates**: Select industry profiles from the dropdown right next to the logo:
  * 💻 **IT / Software Engineer** (Preloads *Vikram Malhotra* with headshot)
  * 🎨 **UI/UX Designer** (Preloads *Ananya Sen* with headshot)
  * 💼 **Senior Project Manager** (Preloads *Aarav Sharma* with headshot)
  * 📣 **Growth Marketer** (Preloads *Rohan Das* with headshot)
* All images are loaded dynamically using high-resolution headshot links!

## 💻 Local Setup & Development

Run this project on your local machine in seconds:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Saumy02/Resume_Builder-ResuSpark.git
   cd Resume_Builder-ResuSpark
   ```

2. **Run a local development server:**
   * If you have python:
     ```bash
     python3 -m http.server 8000
     ```
   * If you have Node.js:
     ```bash
     npx serve .
     ```

3. **Open the browser:**
   Go to `http://localhost:8000` to start designing!
